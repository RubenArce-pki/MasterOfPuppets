# MasterOfPuppets

## Basic to start
- Install node: https://nodejs.org/en
- Install yarn: npm install --global yarn 

## Run

- Download node modules:

```text
    yarn
```

- Execute project:

```text
    yarn dev
```

- Edit people configuration file adding desired names:

```text
    src/data/puppets.json
```

## Build (not necessary for scrums)

- Download node modules:

```text
    yarn build
```
